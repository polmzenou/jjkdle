# Assets

⚠️ **Visuels 100 % originaux uniquement.** Aucune image officielle Jujutsu
Kaisen copyrightée ne doit être ajoutée ici.

- `characters/` — silhouettes/illustrations SVG maison par personnage.
  Convention : `characters/<id>.svg` (l'`id` correspond à `Character.id` dans
  `data/roster/characters.ts`). À défaut, `CharacterCard` affiche les initiales
  du personnage comme placeholder.
- `characters/placeholder.svg` — silhouette générique de secours.
